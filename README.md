# Compact Translator

Ung dung desktop toi gian cho demo va bao cao luong xu ly:

- Audio source: system / microphone / both
- STT: Deepgram realtime
- TTT: Azure Translator one-way
- TTS: Edge hoac Azure
- UI: Main + History, khong co sidebar

## Run

```bash
cd compact-translator
npm install
npm run tauri dev
```
